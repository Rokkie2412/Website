1.Ubuntu 20.04 LTS



Apapun pertanyaannya untuk linux pemula pasti jawabannya ialah Ubuntu, mengapa ubuntu? karena untuk memiliki libary yang luas jika kalian belom bisa move on sama aplikasi windows di ubuntu ini kalian tidak perlu takut karena memiliki banyak sekali libary. Seklain itu juga tampilan ubuntu sangatlah simple dan menarik karena berbasis gnome3.



Download Ubuntu

2.Manjaro Linux



Manjaro linux masuk dalam daftar ini, karena setelah saya mencoba manjaro linux versi GNOME ini membuat saya seperti menggunakan windows,Mac OS, atau Gnome itu sendiri, tampilannya menarik sangat user friendly, selain itu juga manjaro memiliki 3 versi yaitu manjaro KDE, GNOME dan XFCE bisa dipilih sesuai kebutuhan kalian guys. Manjaro juga menggunakan parent linux yaitu Arch linux sehingga manjaro support AUR pada Arch linus sebagai libary tambahan pada linux ini. Namun ada kekurangan pada linux ini yaitu banyak aplikasi pada AUR tidak sebanyak aplikasi pada repository pada linus yang berbasis debian.



Download Manjaro

3.Linux Mint



Linux mint, bagi para penggun windows yang sangat ingin menggunakan linux dengan UI yang mirip dengan windows ini paling cocok karena mengadaptasi tampilan windows, selain itu juga Linux Mint memiliki 3 versi yaitu Cinamon, Mate, dan XFCE bisa dipilih sesuai kebutuhan kalian guys. Linux Mint memiliki parent linux yaitu Ubuntu yang Ubuntu sendiri memiliki parent linux Debian, yang dimana Linux Mint ini menggunakan repository dari Ubuntu yang berasal dari Debian. Linux Mint juga memiliki libary aplikasi yang luas karena memiliki parent linux Ubuntu sehingga tidak perlu takut lagi untuk mencari aplikasi aplikasi pada Linux Mint.



Download Linux Mint

4.Elementary OS



Elementary OS sangat cocok bagi pengguna Mac OS user yang ingin berpindah ke linux, karena elementary OS ini mengadaptasi tampilan dari Mac OS itu sendiri, selain itu juga Elementary OS ini memiliki parent linux Ubuntu yang dimana Ubuntu ini memiliki banyak sekali libary sehingga yang ingin belajar linux menggunakan elemetary OS sangatlah cocok apalagi bagi pengguna Mac OS yang sudah terbiasa dengan tampilan docking apps.



Download Elementary OS

5.Deepin OS



Di daftar terakhir saya ada Deepin OS, Deepin OS ini sempat kontroversial karena banyak yang bilang bahwa linux ini karena buatan dari negara Tiongkok linux ini mencuri data dari pemakaianya yang sebenarnya itu hoax. Deepin OS terbaru mengadaptasi tampilan dari Mac OS big sur, selain itu juga Deepin OS menurut saya memiliki UI yang paling atraktif karena memiliki banyak animasi. Deepin OS sendiri memiliki parent linux Debian OS sehingga aman bagi user baru yang ingin belajar linux karena memiliki repository yang luas.



Download Deepin OS
